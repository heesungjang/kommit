# CLAUDE.md

## gstack

Use the `/browse` skill from gstack for all web browsing. Never use `mcp__claude-in-chrome__*` tools.

Available skills:
- `/plan-ceo-review` — Rethink the problem, find the 10-star product
- `/plan-eng-review` — Lock in architecture, data flow, and edge cases
- `/review` — Find bugs that pass CI but blow up in production
- `/ship` — Automated release: sync, test, push, open PR
- `/browse` — Visual web browsing, inspect UI, take screenshots
- `/qa` — Test app, find bugs, fix them with atomic commits
- `/qa-only` — Testing without code fixes
- `/setup-browser-cookies` — Import sessions from your real browser
- `/retro` — Generate team retrospectives with metrics
